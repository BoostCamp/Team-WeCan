# Yunseop Song

PitchPerfect Project

## 개요

부스트 캠프 첫 프로젝트 앱

![](./static/image_01.png)

## 기능

#### 기본 기능

- Slow
- Fast
- High Pitch
- Low Pitch
- Echo
- Reverb

#### Kick

- 음성 인식
  - Naver 음성인식 API를 사용함
- ProgressBar
  - 오디오가 재생되고 있을 때 얼마나 재생되고 있는지를 나타냄
- 일시정지 / 다시재생
  - 오디오가 재생되고 있을 때 일시정지를 하거나, 다시 재생을 할 수 있음
